# Top Line Tec - Sistema de Gestión Mayorista

Sistema moderno de gestión de inventario y ventas para distribución mayorista de teléfonos móviles.

## 🚀 Tech Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: TailwindCSS 4
- **Backend**: Firebase (Firestore, Auth, Storage, Functions)
- **State Management**: Zustand
- **Data Fetching**: React Query
- **Forms**: React Hook Form + Zod
- **Routing**: React Router v6

## 🏗️ Project Structure

```
src/
├── features/          # Feature-based modules
│   ├── auth/         # Authentication
│   ├── inventory/    # Phone inventory management
│   ├── clients/      # Client management
│   ├── sales/        # Sales & bulk operations
│   └── dashboard/    # Main dashboard
├── components/       # Shared components
├── lib/             # Third-party configurations
├── hooks/           # Custom React hooks
├── types/           # TypeScript type definitions
├── utils/           # Utility functions
└── assets/          # Static assets
```
